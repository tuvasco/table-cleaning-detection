# Прототип системы детекции уборки столиков по видео

## Описание проекта

Разработан упрощенный прототип системы, которая анализирует видео и определяет основные события, связанные с состоянием одного столика:
- стол пустой;
- стол занят;
- подход к столику.

В качестве основы используется детекция людей с помощью YOLOv8 и логика состояния, построенная на анализе зоны выбранного столика.

## Выбор видео
Было выбрано видео №2, т.к. оно было более четким и столик находился крупным планом под камерой, что было более удобным для создания системы.

## Используемые технологии

- Python 3.8+
- OpenCV
- Ultralytics YOLOv8
- Pandas
- NumPy
- Pillow

## Как запустить проект

### 1. Установить зависимости

```bash
pip install -r requirements.txt
```
### 2. Запустить детекцию столика
```bash
python main.py --video video1.mp4
```
## Полученные результаты:
Среднее время между подходом к столику: 65.333 c ( среднее время было сильно уменьшено из-за частого подхода к столу ребенка )
Проблемный кадр находится в репозитории в файле img.png, проблема заключалась в том, что когда голова человека сидящего за столом пропадала за лампой, то стол становился свободным, было решено добавлением переменных которые считали количество кадров, сколкьо человек должен находится в кадре, чтобы стол считался занятым/свободным, также была сильно задушена логика перехода из состояния "стол занят" -> "подход к столику", чтобы избежать ложных срабатываний.
